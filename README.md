# DXL-Coverage
 ## 使用Layout DXL编写的简易统计需求覆盖率的脚本

- 设置模块属性：模块类型，其中
  - “00 需求覆盖率统计为“统计模块”
  - “01~03”为“需求模块”
  - “04~06”为“测试模块”
 ## 需求覆盖率统计模块设计
  - 第一列手动填写模块名字
  - 第二列DXL布局
    - 根据第一列对应单元格的名字，读取对应模块，统计标记为“需求”的对象的个数
  - 第三列DXL布局
    - 根据第一列对应单元格的名字，读取对应模块，统计标记为“需求”的对象的个数
    - 对对应模块激活过滤器，过滤有入向链接的需求，然后统计标记为“需求”的对象的个数
    - 两个个数相除，得到覆盖率
